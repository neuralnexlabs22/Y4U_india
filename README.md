# Y4U INDIA

**Wear Ur Story**

Y4U INDIA is a premium fashion and lifestyle shopping experience built with modern design, immersive animations, and luxury aesthetics.

## Features

- Premium fashion-focused UI
- Responsive design
- Smooth animations and transitions
- Modern product showcase
- Luxury beige and brown theme
- Mobile-first experience

## Tech Stack

- React
- TypeScript
- Vite
- CSS Modules
